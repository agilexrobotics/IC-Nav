- ### 1.2.4. 声呐设备栅格化数据

  - 声呐数据根据地图栅格化之后的数据，配置几个声呐会有几个声呐的frame_id

  - GET请求　/gs-robot/real_time_data/ultrasonic_phit

  - response:

    ```
    {
      "header":{
        "stamp":123123,
        "frame_ids":["ultrasonic0","ultrasonic1","ultrasonic3"]
      },
      "mapInfo":{　//地图信息
        "gridWidth":1000,　//栅格地图宽度
        "gridHeight":1000, //栅格地图高度
        "resolution":0.05　//栅格化地图比例　
      },
      "gridPhits":[
        {x:111,y:111},{x:111,y:111},{x:111,y:111}　
      ], // 栅格化的数据，如过有２个超声，数据会有４个，按顺序依次排列，每两个的第一个代表超声的原点，第二个代表探测到的障碍物位置
      "worldPhits":[
        {
            "x": -0.00000000000180135178,
            "y": 0.00000002584042026,
            "z": 0
        },
        {
            "x": -0.00000000000180135178,
            "y": 0.00000002584042026,
            "z": 0
        },
        {
            "x": -0.00000000000180135178,
            "y": 0.00000002584042026,
            "z": 0
        }
      ]// 原始数据，如过有２个超声，数据会有４个，每两个的第一个代表超声的原点，第二个代表探测到的障碍物位置
    }
    ```
